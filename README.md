# Manish Kumar Portfolio

A simple portfolio website built with HTML, CSS, and JavaScript. The project includes a landing page, a contact page, animated background effects, and interactive hero card behavior.

## Features

- Responsive hero section with profile card and call-to-action buttons
- Animated background using `particles.js`
- Hover and cursor-driven interactions for cards and hero profile
- Contact form layout ready for integration
- Stylish glassmorphism-inspired UI with gradient accents
- Optional React component in `AnimatedPortfolio.jsx` for React-based versions

## Files

- `index.html` — main portfolio landing page
- `contact.html` — contact page with form and hero section
- `style.css` — shared stylesheet for visuals and layout
- `script.js` — interactive behavior and animation logic
- `AnimatedPortfolio.jsx` — standalone React component version for React apps
- `resume.pdf` — downloadable resume asset
- `image/` — image assets used by the site

## Getting Started

1. Open `index.html` or `contact.html` in a browser.
2. Or use a local server for best results, for example:
   - `npx http-server .` (requires Node.js)
   - or use VS Code Live Server extension
3. The page automatically loads `style.css` and `script.js` for styling and interaction.

## Notes

- `contact.html` currently links the local `style.css` and `script.js` files.
- `AnimatedPortfolio.jsx` is a React-friendly version of the same portfolio UI and can be used in a React app build.
- The project uses `particles.js` for the animated background and can be customized in `script.js`.

## Customize

- Update `index.html` and `contact.html` content to match your personal details.
- Replace the profile image in `image/hero.jpg` with your own photo.
- Edit `style.css` to change colors, spacing, and typography.
- Modify `script.js` to adjust animations, pointer effects, and page interactions.
